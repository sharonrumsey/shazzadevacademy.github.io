Explain to your non-tech friend -- Describe what these functions do in your own words:

.map()

This method creates a new array with the results of calling a provided function on every element in this arrary

Map processes through each item in an array or objects and calls back or returns a value(s) as a new array. Particularly usefull when you wish to return values from specific inputs or object lists. Callback - Function that produces an element of the new Array, taking three agruments (currentValue, index, array).

.filter()

Creates a new array with all elements that pass the test implemented by the provided function.

It goes through the array and pulls back a value of a new array showing true for the argument. (currentValue, index, array)

.reduce()

This method applies a function against an accumulator and each value of the array (from left to right) to reduce it to a single value.

Goes through the array taking the argument and brings back a single value. (currentValue, index, array)
