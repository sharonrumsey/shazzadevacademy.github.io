What is grid based design?

Grid based design is see the page as a grid for setting the design within the grid as a structure.

What's all the hype about responsive webpages?

The hype is so all devices that access your website will function in way that works best for the size of that device.

Semantic Structure

“Semantic elements are elements with a meaning. It describes the meaning to both the browser and developer.”

So, you should use the elements that best describes the content of what you are putting into the webpage.

Any other points you found interesting and want to reflect on?
